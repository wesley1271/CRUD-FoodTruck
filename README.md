# 🚚🍔 Food Truck Virtual - CRUD de Pedidos

### Sistema simples em Python para gerenciamento de pedidos de um Food Truck, utilizando conceitos fundamentais da programação como CRUD, manipulação de arquivos e estruturas de controle.

---

## 📌 Funcionalidades

O sistema funciona via terminal e permite:

 - **Create (Criar)** - Adicionar novos pedidos;
 - **Read (Ler)** - Visualizar pedidos cadastrados;
 - **Update (Atualizar)** - Editar pedidos existentes;
 - **Delete (Excluir)** - Remover pedidos;
 - **Sair** - Encerra o sistema e limpa os pedidos;

---
 
## 🗂️ Estrutura do Projeto

---

### 📁 projetoCRUD

- **Food_truck.py** | Arquivo principal com toda lógica do sistema
- **pedidos.txt** | Banco de dados simples (armazenamento dos pedidos)
- **nome.txt** | (Arquivo auxiliar, se utilizado)
- **README.md** | Documentação do projeto

---

## 🧠 Conceitos aplicados

- Manipulação de arquivos (open, read, write);
- Estruturas condicionais (if, elif, else);
- Laços de repetição (while, for);
- Tratamento de exceções (try/except);
- Organização de dados em formato texto (simulando banco de dados);
